# Birthday-Blast
Welcome to the official website of Birthday Blast, an interactive birthday countdown featuring confetti animations, a customizable terminal, and a secure digital time capsule.

## Demo

Check out this site live at: [Live Site](https://rainier-ps.github.io/Birthday-Blast/)!

## Usage

* Clone the repository
* Open `index.html` in a browser
* Edit HTML/CSS for customization
